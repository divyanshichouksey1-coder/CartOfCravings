# CartOfCravings - The Addiction Starts at 4:30 PM

A maximalist & premium single-page React application showcasing street food luxury from Lucknow.

## Features

- **Dark Gastronomy Theme**: Pitch-black background with Neon Amber and Electric Orange accents
- **Hero Section**: Massive headline with custom fonts (Bebas Neue & Playfair Display)
- **Parallax Effects**: Floating and rotating food images on scroll
- **Interactive Menu**: Hover effects with glassmorphism styling
- **Live Status Counter**: Real-time countdown to opening time (4:30 PM) or "Sizzling Now" indicator
- **Instagram Integration**: Direct link to @cartofcravings.coc
- **Location Map**: Dark mode map showing the cart location
- **Mobile Optimized**: Floating call button for mobile devices

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
